### ‚öôÔ∏è Configuration Options

#### Network Configuration

```yaml
# Network Topology
topology:
  type: Star
  devices: 50+
  vlans: 10
  subnets: 8

# DHCP Configuration
dhcp:
  enabled: true
  pool_size: 1000
  lease_time: 24h

# DNS Configuration
dns:
  primary: 8.8.8.8
  secondary: 8.8.4.4
  local: 192.168.1.1
```

```yaml
  humidity: 6 devices

  security: 2 devices

port: 1883
```
